# Net-Craft Studio

A highly performant, premium web development agency portfolio built with modern React. It showcases buttery-smooth scroll animations, dynamic page transitions, and an editorial design aesthetic.

## 🚀 Technologies

*   **React 18**: Component-based architecture with React Router for seamless navigation.
*   **GSAP & ScrollTrigger**: State-of-the-art scroll animations, complex timeline scrubbing, and pinning.
*   **Lenis**: Hardware-accelerated smooth scrolling integrated directly with GSAP's ticker for zero lag.
*   **Tailwind CSS**: Rapid, scalable, and responsive utility-first styling.
*   **Canvas API**: Used for rendering high-resolution, buttery-smooth 300+ frame video sequences tied strictly to scroll position without DOM overhead.

## ✨ Key Features

*   **Cinematic Hero Section**: A scroll-linked canvas image sequence that shrinks, pins, and expands into a flawless full-screen video experience.
*   **Progressive Client Portal**: An interactive, dynamic form that builds up step-by-step inline without jarring page refreshes.
*   **Dynamic Navbar**: Intelligently hides when reading content and smartly reappears when scrolling backwards into specific, permitted sections to keep the user immersed.
*   **Performant Marquees**: CSS-driven infinite marquees for brand logos.
*   **Zero-Jitter Transitions**: Advanced GSAP state-tracking to prevent layout thrashing and animation conflicts.

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
